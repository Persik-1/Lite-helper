Внешний вид (CSS)

CSS можно сделать и в файле html, но скорее всего, если вы будете делать так, то у Вас в html будет слишком большой код, поэтому создаём отдельный файл, а в html под "title" пишем:
link rel="stylesheet" href="(название файла для CSS)"
и заключаем эту строчку в <>

В CSS можно менять как сами блоки, по типу "p"(простой шрифт), так и создавать отдельные классы и теги и применять их к блокам, благодаря чему, мы можем создать разнообразный сайт

Дополнительные функции и веб-штучки (JavaScript/js)

JavaScript используется для создания анимаций, функций и многого другого, при желании можно будет даже создать мини-игру у себя на сайте. Довольно сложно будет объяснить все возможности JavaScript'а, посему, я могу сказать Вам лишь как добавить JavaScript на страницу и посоветовать сайты с информацией

Для того, чтобы добавить себе на сайт Java Script, Вам для начала надо создать отдельный файл для него, оканчивающийся на .js, а после, в файле .html прямо перед блоком body вставить код:
src="(название файла с JavaScript).js"
и закрывающий его блок:
/script
Оба должны заключены в <>
