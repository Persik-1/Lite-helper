Начало (HTML)

Для начала, рекомендую скачать Visual Studio Code , с её помощью будет проще кодить

После чего запустите приложение, выберите папку, вкоторой будет Ваш
код, в ней создайте файлы index.html, style.css и script.js

В index.html введите "!" и нажмите "Tab", это создаст начальный код
